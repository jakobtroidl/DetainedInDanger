# Detained in Danger

## Prerequisites
* internet connection
* install node.js and npm. You can downlond both [here](https://www.npmjs.com/get-npm)

## Get started
* navigate to the `/humanrightsfirst-master` folder
* run `npm install`
* run `node app.js`
* you can access the webpage under `http://localhost:5000/` in your browser
